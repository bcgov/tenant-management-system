# What is CSTAR?

CSTAR stands for Connected Services Team Access and Roles.

CSTAR provides a common way to manage tenant-based access for connected services. It keeps track of tenants, the users who belong to those tenants, the groups those users are part of, and the roles that connected services can use when deciding what a user is allowed to do.

In practical terms, CSTAR gives connected services a shared access-management layer. A service team does not need to build its own tenant, user, group, and role management system from scratch if CSTAR can provide that information.
